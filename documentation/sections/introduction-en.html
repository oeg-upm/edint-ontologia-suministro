Introduction back to ToC

This ontology is part of the catalog of ontologies that is being developed in the context of the Data Space for Smart Urban Infrastructures (EDINT). Likewise, in the energy consumption domain, it was identified as a relevant ontology for the representation of data managed by local entities in the guide FEMP Open Data 2019: 40 datasets to be published by Local Entities, as also described in the corresponding data.gob.es note, and therefore included in the Github repository supporting the initiative, which is deprecated once this repository is launched.
